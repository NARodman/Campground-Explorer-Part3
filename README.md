# Android Lab 6 - Campground Explorer Pt. 3

Submitted by: **Nina Lanier**

**Campground Explorer Pt. 3** is an Android app that combines the Parks and Campgrounds applications from Units 3 and 4. Users can browse National Park Service parks and campgrounds, switching between the two using a bottom navigation bar. The app uses fragments for dynamic UI updates and RecyclerView to display lists efficiently.

Time spent: **1.5 hours** spent in total

## Required Features

The following **required** functionality is completed:

- [x] **Add and implement Bottom Navigation to switch between screens**
- [x] **Dynamically use Fragments within Activities for Parks and Campgrounds**
- [x] **Customize the toolbar and icons**
